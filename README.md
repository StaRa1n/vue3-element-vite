### 安装使用

##### 获取代码

```
git clone https://github.com/StaRa1n/vue3-element-vite.git
```

##### 安装依赖

```
cd vue3-element-vite
pnpm install
```

##### 运行

```
pnpm run dev
```

### 项目结构

```

```
